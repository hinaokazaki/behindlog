# Behindlog — 採用担当向けアプリ概要

## アプリの概要

**一言で言うと：** 目標に向けた日々の学習・作業を記録し、友達と共有して継続をサポートするウェブアプリです。

「毎日コツコツやる努力を可視化し、仲間と続けやすくする」というコンセプトで開発しました。

---

## 主な機能

| 機能 | 内容 |
|------|------|
| **ダッシュボード** | 目標・Todo・累計学習時間・カレンダーを一画面で確認 |
| **毎日の記録** | 日次のTodo消化・学習時間（Committime）を入力 |
| **コミュニティ** | 友達をメール招待し、互いのダッシュボードを閲覧・応援 |
| **目標管理** | 長期ゴールを設定し、日々の記録と紐付けて進捗確認 |

---

## 技術スタック

- **フロントエンド:** Next.js (App Router) / TypeScript / Tailwind CSS
- **認証:** Supabase Auth（メール認証 + ゲストログイン）
- **DB/ORM:** PostgreSQL (Supabase) + Prisma
- **データフェッチ:** SWR
- **バリデーション:** Zod
- **デプロイ:** Vercel

---

## アピールポイント

- ゲストログインで採用担当者がすぐ触れる設計
- フレンドシップ機能（招待→承認フロー）を自前実装
- レスポンシブ対応（モバイル/タブレット/大画面で異なるレイアウト）
- Supabase認証とPrismaを組み合わせたフルスタック構成
